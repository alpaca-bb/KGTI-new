# KGTI - 港科广专属人格测试
HKUST(GZ) Original Campus Personality Test

KGTI 是一款专为香港科技大学（广州）学生设计的原创校园人格测试，结合学生日常、性格特质、行为习惯生成专属人格画像与可爱小鸟形象，纯前端网页实现，开箱即用。


##  项目特色
- 专为 HKUST(GZ) 校园文化定制
- 18 道测试题 → 自动计算人格结果
- 专属原创小鸟形象（卷心菜、DDL战神、深夜哲人等）
- 纯 HTML + CSS + JavaScript 实现
- 无需后端，GitHub Pages 一键部署
- 响应式设计，手机 / 电脑均可打开测试
